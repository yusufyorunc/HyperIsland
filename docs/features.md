# 功能介绍

HyperIsland 为澎湃 OS3 提供了丰富的超级岛通知增强功能，让你的通知体验更加现代化。

## 软件功能

### 应用适配

支持为任意应用启用超级岛功能，每个应用可单独配置。

- **搜索应用**：快速搜索应用名或包名
- **单独开关**：每个应用独立控制是否启用
- **批量管理**：一键查看已启用的应用数量

![应用适配](./images/Screenshot_2026-04-05-00-00-06-698_io.github.hype.jpg){style="width: 50%;"}
### 通知渠道管理

针对支持多通知渠道的应用（如 QQ），可以为不同渠道单独配置：

- **即时消息**：聊天通讯类通知
- **系统推送**：MiPush推送通知
![渠道设置](./images/Screenshot_2026-04-05-00-00-04-279_io.github.hype.jpg){style="width: 50%;"}
每个渠道可独立设置模板和样式。

## 超级岛自定义

### 模板选择

为每个应用/渠道选择合适的超级岛模板：

| 模板 | 说明 |
|:-----|:-----|
| 通知超级岛 | 支持任意通知转为焦点通知 + 超级岛显示 |
| 通知超级岛-精简 | 自动去除通知中的 "x条新消息" 和重复字段 |
| 下载 | 自动识别下载状态并转为焦点通知 + 超级岛 |
| 下载-Lite | 超级岛只显示图标 + 进度圈 |
| AI 通知超级岛 | 超级岛左右交给 AI 精简 |
| 模板 | 说明 |

### 样式选择

| 样式             | 说明                   |
|:---------------|:---------------------|
| 新图文组件 + 底部文本按钮 | 底部文本按钮，最多支持2个按钮      |
| 封面组件 + 自动换行    | 支持2行显示的焦点通知，配有底部文本按钮 |
| 新图文组件 + 右侧文本按钮 | 右侧文本按钮，仅支持一个按钮       |

## 岛自定义

- **超级岛图标**：自动或自定义选择
- **大岛图标**：是否显示大岛图标
- **初次展开**：通知首次显示时是否自动展开
- **更新展开**：通知更新时是否自动展开
- **消息滚动**：超级岛内文字是否滚动显示
- **自动消失**：设置超级岛自动消失的秒数
- **高亮颜色**：自定义超级岛高亮颜色（支持 HEX 色值）
- **文本高亮**：选择左侧或右侧文本以高亮颜色显示

## 焦点通知自定义

- **焦点图标**：选择焦点通知面板中显示的图标
  - 自动：使用应用默认图标
  - 自定义：手动选择图标
- **焦点通知**：控制焦点通知的显示方式
  - 默认（开启）：正常显示焦点通知
  - 关闭：通知还原为普通通知，仅显示超级岛
- **状态栏图标**：是否在状态栏显示图标
- **锁屏通知复原**：锁屏时是否恢复为普通通知样式，以使用系统自带的隐私管理策略

::: warning
关闭焦点通知时，超级岛由**系统界面**代发，可能有兼容性问题
:::
### 焦点通知破解

::: danger 内置破解
软件内置破解白名单功能，但不支持安全模式，可能导致系统界面无限崩溃，请确保有能力救砖再启用。
:::

通过 HyperCeiler 或内置破解，可以：
- 移除焦点通知白名单限制
- 解锁焦点通知白名单验证
- 让任意应用的通知都能以焦点通知形式显示

## 下载管理器拓展

拦截 HyperOS 下载管理器的通知，以超级岛样式展示文件名与下载进度。

::: tip 核心能力
- 支持**暂停**、**继续**、**取消**操作
- 暂停后显示一条恢复下载的通知（需要启用下载管理器Hook）
:::

::: tip 启用方式
下载上岛默认关闭，需要自行去应用中开启 **「显示系统应用」** 并勾选 **「下载管理程序」**。
:::
